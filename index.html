What is Fujihack?

  • Fujihack is an open-source hacked utility that can be run on Fujifilm cameras.
  • Fujihack can be run with a patched firmware. Other methods are being worked on and researched.
  • It's currently a research project with the goal of improving and repurposing cameras.

What's Been Done?

  • Unpack, repack, and modify firmware
  • Harmless code execution hijacks in firmware
  • Portable debugger firmware patch with brute force RAM searching
  • Patch PTP/USB opcode to run custom code and experiment quickly over USB
  • Uncovered hidden scripting abilities
  • Complete RAM and flash dump
  • Interface with embedded internal SQLite
  • Extend 14 minute record limit
  • Read IO: buttons, gyroscope, accelerometer
  • Fix broken shutter button with clever button remapping
  • Draw debug text and rasterizer objects
  • OpenGL graphics buffer rendering
  • Background task creation, deletion, semaphores, task hooks, events
  • Primitive card speed tests (capped at 35mbps)
  • Gryoscope Tetris, 7fps DOOM port
  • POSIX and Frontier compliant compatibility layer

Hacked Models

  • X-A2 @ 1.30 (Bin loader, USB code runner)
  • XF1 @ 1.01 (USB code runner)
  • Firmware downgrade has been tested on some lenses. (same firmware format used)

What's being worked on?

  • Gain stable code execution on the camera without firmware patching
  • Creating a custom menu with apps, scripts, and real time patching and tweaking
  • Reverse engineering and tweaking things like film modes, video, and imaging code

Would it work with my camera?

FujiHack will only work on Fujifilm X/GFX/HS/F/Z/Pro cameras. You can put your firmware file into the web patcher to see if FujiHack is compatible with your firmware type. In general, FujiHack does NOT support the XP and S series Fujifilm cameras. The S Series has already been hacked by CHDK forum members.

What is it written in?

The firmware debugger patches are written in Standard ARM Assembly. Most Fujifilm cameras run a 32 bit ARM v6 SoC. Everything else is written in C, with some apps and scripts being written in Rust and Javascript.

When will you work on X?

This is an open-source non-profit project, and currently is not backed by any company, and has no deadlines or plans. Consider it a miracle that it exists in the first place.
